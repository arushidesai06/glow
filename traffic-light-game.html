⚡ ARDUINO SETUP

⚠️ Web Serial only works in Chrome or Edge. The game must be opened via a local server, NOT double-clicked from Finder.

STEP 1 — SERVE THE FILE LOCALLY

Open Terminal on your Mac and run:

cd ~/Downloads python3 -m http.server 8000

Then open Chrome and go to: http://localhost:8000/traffic-light-game.html

STEP 2 — ARDUINO SKETCH

Upload this to your Elegoo Uno via Arduino IDE:

const int RED_PIN = 13; const int YELLOW_PIN = 12; const int GREEN_PIN = 11; void setup() { Serial.begin(9600); pinMode(RED_PIN, OUTPUT); pinMode(YELLOW_PIN, OUTPUT); pinMode(GREEN_PIN, OUTPUT); // All off at start digitalWrite(RED_PIN, LOW); digitalWrite(YELLOW_PIN, LOW); digitalWrite(GREEN_PIN, LOW); } void loop() { if (Serial.available() > 0) { char cmd = Serial.read(); digitalWrite(RED_PIN, LOW); digitalWrite(YELLOW_PIN, LOW); digitalWrite(GREEN_PIN, LOW); if (cmd == 'R') digitalWrite(RED_PIN, HIGH); else if (cmd == 'Y') digitalWrite(YELLOW_PIN, HIGH); else if (cmd == 'G') digitalWrite(GREEN_PIN, HIGH); // cmd == 'O' = all off } }

STEP 3 — CONNECT

  • Close Arduino IDE Serial Monitor (it blocks the port)
  • Click "Connect Arduino" in the game
  • Select your port (e.g. /dev/cu.usbmodem...)
